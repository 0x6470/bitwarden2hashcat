# bitwarden2hashcat
A tool that converts Bitwarden's data into a hashcat-suitable hash

## Requirements
`Python >= 3.7`

`plyvel module`

## Installation
### Windows
`python -m pip install -r requirements_windows.txt`

### Linux
`pip3 install -r requirements_linux.txt`
